# software-testing-learning

这个仓库用来记录我系统学习 **软件测试（Software Testing）** 的全过程，发布出来仅为了自我鞭策。  
从基础理论到实战项目，从人工测试到自动化测试，希望通过这段旅程，  
逐步建立起自己在**软件质量保障与测试工程**方向的系统能力。

---

## 学习目标

- 掌握软件测试的基本概念、流程与方法论  
- 能独立编写测试用例，执行功能与性能测试  
- 熟悉常见的自动化测试框架（如 Pytest、Selenium、Postman 等）  
- 了解持续集成（CI/CD）中的测试实践  
- 最终完成一个完整的测试项目，用以展示学习成果  


---
